Css = cascading style sheet

các cách đưa css vào website

  1. c1: inline - viết vào thuộc tính style trong thẻ mở của html
  2. c2: internal - viết trong cặp thẻ style ở phần head
  3. c3: external - viết vào file.css và nhúng vào tài liệu html thông qua thẻ link:css
